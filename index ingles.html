Resident Security Program (RSP)

What is RSP?

The RSP Project is a program created for the security of a residence with the purpose of providing comfort and security to the residents of the area, using the main service and development tools such as Visual Studio, which allows to automate the compilation in source code..

How does it work?

The first window when starting the system is the Log in, where you have to enter the credentials, which is registered in the database, then the menu will be displayed containing six form options. Each one will have its respective function, for the different needs that the residents have, thus being able to provide security through programming tools.
