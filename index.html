Resident Security Program (RSP)

Resident Security Program (RSP)

¿Qué es RSP?

El proyecto RSP es un programa creado para la seguridad de una residencia con el propósito de brindar comodidad y seguridad a los residentes de la zona, mediante las principales herramientas de servicio y desarrollo como Visual Studio, que permite automatizar la compilación en código fuente.

¿Como funciona?

La primera ventana al iniciar el sistema es el Log in, donde se tiene que ingresar las credenciales, que se tiene registrada en la base de datos, luego se mostrara el menu que contendra seis opciones de formularios. Cada uno tendra su respectiva funcion, para las diferentes necesidades que tienen los residentes , asi siendo capaz de brindar seguridad por medio de herramientas de programación

Preguntas frecuentes

Desconfiguración de resolución por pantallas muy grandes
Un programa como este es algo vital para las residenciales que hoy en día no cuenta con la seguridad que sus residentes necesitan, debido a la delincuencia del país se exige mayor nivel de seguridad. Este es un programa con el cual se puede llevar un total de registro de los propietarios de cada casa, a su vez también se lleva el registro de toda persona ajena a la residencial que entra y sale del lugar, debido a que se marca la hora a la que accede al lugar y su hora de salida.
Existen muchos métodos de encriptación, pero en este caso nuestro proyecto cuenta con la encriptación de contraseñas MD5 uno de sus usos es el de comprobar que algún registro no haya sido modificado.
